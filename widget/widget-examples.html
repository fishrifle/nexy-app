üì¶ Next.js Integration

Add the widget script to your Next.js app:

// In your component or layout import Script from 'next/script' export default function Layout({ children }) { return ( <> {children} <Script src="/widget/sponsorship-widget.js" /> </> ) } // Then use the widget anywhere: <div data-sc-widget="sponsorship"></div>

üîß WordPress Integration

Add to your theme's functions.php:

function enqueue_sponsorship_widget() { wp_enqueue_script( 'sponsorship-widget', get_template_directory_uri() . '/js/sponsorship-widget.js', array(), '1.0.0', true ); } add_action('wp_enqueue_scripts', 'enqueue_sponsorship_widget'); // Use shortcode in posts/pages: [sponsorship_widget amount="50"] // Or in templates: <div data-sc-widget="sponsorship"></div>

üåê Drupal Integration

Add to your theme's .libraries.yml and templates:

# In your_theme.libraries.yml sponsorship-widget: js: js/sponsorship-widget.js: {} # In your template file: {{ attach_library('your_theme/sponsorship-widget') }} <div data-sc-widget="sponsorship"></div> # Or create a custom block plugin
