üöÄ Quick Start

Get up and running with the sponsorship widget in under 5 minutes:

1. Include the Script

<script src="/widget/passiton-embed.js"></script>

2. Add the Widget

<div id="passiton-donate-widget"></div>

3. Configure (Optional)

window.PassItOnConfig = { defaultAmount: 50, buttonText: 'Sponsor a Developer', color: '#3b82f6', position: 'bottom-right' };

Customize the widget behavior and appearance:

Property Type Default Description
targetElementId string passiton-donate-widget ID of the element where widget will be rendered
organizationId string '' Your organization identifier
defaultAmount number 25 Default donation amount in dollars
color string #3b82f6 Primary color for the widget theme
buttonText string Donate Text displayed on the main button

üîß Advanced Usage

Multiple Widgets

You can have multiple widgets on the same page:

<div id="widget-header"></div> <div id="widget-sidebar"></div> <script> // Initialize first widget PassItOnWidget.init({ targetElementId: 'widget-header', defaultAmount: 25, buttonText: 'Quick Donation' }); // Initialize second widget PassItOnWidget.init({ targetElementId: 'widget-sidebar', defaultAmount: 100, buttonText: 'Major Gift', color: '#10b981' }); </script>

Event Handling

Listen for widget events:

window.addEventListener('message', function(event) { if (event.origin !== 'https://donation-one-phi.vercel.app') return; switch(event.data.type) { case 'donation_complete': console.log('Donation completed!', event.data); // Track analytics, show thank you message, etc. break; case 'close_modal': console.log('Widget closed'); break; } });

üõ†Ô∏è Troubleshooting

Widget Not Appearing

  • Ensure the script is loaded before DOM content
  • Check that the target element ID exists
  • Verify there are no JavaScript errors in console

Styling Issues

  • Widget uses high z-index (9999) for positioning
  • Modal overlay has z-index 10000
  • Customize colors using the color config option

HTTPS Requirements

  • Widget requires HTTPS for payment processing
  • Cross-origin messaging works only with secure contexts
